### Задание 1: Чтение и запись текста в файл

1. Напишите программу, которая:
   - Запрашивает у пользователя ввод строки.
   - Сохраняет введённую строку в текстовый файл (output.txt).
   - После этого программа должна прочитать содержимое файла и вывести его на экран.

### Задание 2: Работа с CSV

2. Создайте программу, которая:
   - Создает CSV файл (students.csv) и записывает в него информацию о студентах.
    Данные должны содержать следующие поля: имя, возраст, специальность.
   - Затем программа должна загрузить данные из CSV файла и вывести информацию о каждом студенте на экран.

### Задание 3: Работа с JSON

3. Напишите программу, которая:
   - Создает JSON файл (data.json), который будет содержать информацию о книгах.
    Каждая книга должна иметь следующие поля: название, автор, год издания и цена.
   - Затем программа должна загрузить данные из JSON файла и вывести информацию о каждой книге на экран.

### Задание 4: Обработка ошибок при работе с файлами

4. Создайте программу, которая:
   - Пытается открыть и прочитать файл (non_existent_file.txt).
   - Обработайте возможное исключение и выведите сообщение, информирующее пользователя о том, что файл не найден.
   - Включите возможность создания файла с таким именем и записи в него сообщения, если файла не существует.

### Задание 5: Сравнение данных

5. Напишите программу, которая:
   - Загружает данные из двух CSV файлов (file1.csv и file2.csv), которые содержат информацию о клиентах (имя, телефон).
   - Сравнивает данные из обоих файлов и выводит на экран имена клиентов, которые присутствуют в обоих файлах.

### Задание 6: Изменение данных (CSV и JSON)

6. Создайте программу, которая:
   - Загружает данные из JSON файла (inventory.json),
    в котором содержится информация о товарах (название, категория, цена).
   - Позволяет пользователю обновить цену товара по его названию.
    После обновления, программа должна сохранить изменения обратно в файл.

### Задание 7: Фильтрация данных

7. Напишите программу, которая:
   - Загружает данные из CSV файла (products.csv), где хранятся данные о товарах (название, цена).
   - Фильтрует товары по заданной минимальной цене, введенной пользователем, и выводит отфильтрованные результаты.
