### Задание 1: Основы NumPy

1. Напишите программу, которая:
   - Создает массив NumPy из 10 случайных целых чисел от 1 до 100.
   - Выводит на экран среднее, максимальное и минимальное значения этого массива.

### Задание 2: Основы Pandas

2. Создайте программу, которая:
   - Создает DataFrame с информацией о студентах с такими столбцами, как Имя, Возраст, Специальность,
    Оценка (случайные данные).
   - Выводит на экран DataFrame и затем выводит только тех студентов, у которых оценка выше 75.

### Задание 3: Работа с CSV и Pandas

3. Напишите программу, которая:
   - Загружает данные из CSV файла (sales_data.csv) с информацией о продажах с такими столбцами, как Дата,
    Покупатель, Сумма.
   - Подсчитывает и выводит общую сумму продаж за каждый месяц.

### Задание 4: Визуализация данных с Matplotlib

4. Создайте программу, которая:
   - Загружает данные из CSV файла (expenses.csv), в котором содержится информация о расходах
    (категория и сумма).
   - Строит круговую диаграмму, показывающую распределение расходов по категориям с помощью Matplotlib.

### Задание 5: Сравнение данных с Pandas

5. Напишите программу, которая:
   - Создает два DataFrame: df1 (информация о продаже) и df2 (информация о возврате).
   - Сравнивает эти DataFrame и выводит только те элементы, которые были проданы, но не были возвращены.

### Задание 6: Агрегация данных с Pandas

6. Создайте программу, которая:
   - Загружает данные о продажах из CSV файла (sales.csv), содержащего столбцы Товар, Количество, Цена.
   - Подсчитывает общую выручку для каждого товара и выводит результаты в виде нового DataFrame.

### Задание 7: Графики с Matplotlib

7. Напишите программу, которая:

   - Создает гистограмму этих чисел с помощью Matplotlib, чтобы показать распределение значений.

### Задание 8: Временные ряды с Pandas

8. Создайте программу, которая:

### Задание 9: Статистический анализ с NumPy и Pandas

9. Напишите программу, которая:
   - Загружает набор данных о результатах тестирования студентов из CSV файла
    (test_scores.csv) с такими столбцами,
    как Имя, Оценка.
   - Использует Pandas для нахождения медианной оценки и стандартного отклонения,
    а NumPy для других статистических расчетов, таких как среднее и диапазон.
