### Задания по ООП

#### Задание 1: Создание классов и объектов
1. Создайте класс Car, который будет содержать атрибуты: make (марка), model (модель) и year (год выпуска).
   - Добавьте метод display_info(), который выводит информацию о машине.
   - Создайте несколько объектов класса Car и выведите их информацию на экран.

#### Задание 2: Наследование
1. Создайте базовый класс Animal с методами speak() и info().
   - Метод speak() должен выводить звук, который издает животное.
   - Метод info() должен выводить информацию о животном.

2. Создайте подклассы Dog и Cat, наследующие класс Animal и реализующие свои собственные версии методов speak() и info().
   - В классе Dog speak() должен выводить "Гав", а info() — информацию о собаке.
   - В классе Cat speak() должен выводить "Мяу", а info() — информацию о кошке.

#### Задание 3: Инкапсуляция
1. Создайте класс BankAccount, содержащий атрибуты account_number (номер счета) и balance (баланс).
   - Сделайте атрибут balance защищенным (private) с помощью двойного подчеркивания.
   - Добавьте методы deposit(amount) и withdraw(amount), которые изменяют баланс.
    Обеспечьте, чтобы нельзя было снять больше средств, чем доступно на счете.
   - Создайте объект BankAccount и протестируйте методы.

#### Задание 4: Полиморфизм
1. Создайте базовый класс Shape с методом area().
2. Создайте два подкласса: Rectangle и Circle, которые реализуют свой собственный метод area().
   - В классе Rectangle метод area() должен возвращать площадь прямоугольника по формуле width * height.
   - В классе Circle метод area() должен возвращать площадь круга по формуле π * radius^2.
3. Напишите функцию, которая принимает список форм и выводит площадь для каждой из них,
 независимо от того, является ли фигура кругом или прямоугольником.

#### Задание 5: Сложение объектов
1. Реализуйте класс Vector, который будет представлять вектор в пространстве с координатами (x, y).
   - Реализуйте методы для сложения двух векторов и получения длины вектора.
   - Перегрузите оператор + для сложения объектов класса Vector.
