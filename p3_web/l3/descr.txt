### Задание 1: Основы SQL

1. Напишите SQL-запросы для выполнения следующих действий на созданной вами таблице students с такими столбцами,
 как id, name, age, grade:
   - Создайте таблицу students.
   - Вставьте в таблицу 5 записей о студентах.
   - Выберите всех студентов с оценкой выше 70.
   - Обновите возраст одного из студентов.
   - Удалите студента с конкретным id.

### Задание 2: Создание базы данных SQLite

2. Создайте SQLite базу данных с таблицей books с такими столбцами, как id, title, author, published_year.
 Напишите программу на Python, которая:
   - Создает базу данных и таблицу.
   - Вставляет 10 записей о книгах.
   - Запрашивает и выводит на экран все записи из таблицы.

### Задание 3: Использование SQLAlchemy

3. Напишите программу на Python, которая:
   - Использует SQLAlchemy для создания базы данных SQLite library.db и таблицы authors с полями
    id, name, birth_date.
   - Использует ORM для добавления 5 авторов, а затем выводит их на экран.

### Задание 4: Запросы с SQLAlchemy

4. Создайте приложение, которое:
   - Использует SQLAlchemy для создания базы данных movies.db и таблицы movies с полями
    id, title, director, release_year.
   - Напишите функции для добавления, обновления и удаления фильмов, а затем протестируйте их.

### Задание 5: Связи между таблицами

5. Воспользуйтесь SQLAlchemy для создания базы данных с двумя связанными таблицами:
   - students (id, name) и enrollments (id, student_id, course_name).
   - Реализуйте связь один-ко-многим между студентами и курсами.
   - Добавьте студентов и курсы, а затем выведите на экран всех студентов и их курсы.

### Задание 6: SQL-запросы с фильтрацией и сортировкой

6. Напишите программу на Python, которая:
   - Использует SQLite для создания таблицы sales с полями id, product_name, quantity, price.
   - Вставляйте данные о продажах, а затем выполните запрос, который возвращает список товаров,
    отсортированных по количеству продаж (от большего к меньшему).

### Задание 7: Обработка ошибок базы данных

7. Создайте программу с использованием SQLAlchemy, которая:
   - Реализует базу данных с таблицей customers, может обрабатывать уникальность поля email.
   - При попытке добавить клиента с существующим email выводит соответствующее сообщение об ошибке.

### Задание 8: Загрузка данных из CSV в базу данных

8. Напишите программу, которая:
   - Загружает данные о студентах из CSV файла (students.csv) и добавляет их в таблицу students в базе данных SQLite,
    используя SQLAlchemy.
   - CSV файл должен содержать колонки name, age, grade.
