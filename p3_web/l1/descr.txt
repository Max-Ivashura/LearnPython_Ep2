### Задание 1: Основы HTTP

1. Напишите простую программу на Python, которая:
   - Отправляет GET-запрос к публичному API
   (например, к [JSONPlaceholder](https://jsonplaceholder.typicode.com/)).
   - Выводит на экран статус-код ответа и заголовки ответа.

### Задание 2: Простой REST API с Flask

2. Создайте простой RESTful API с использованием Flask, который будет:
   - Иметь следующие конечные точки:
     - GET /items – возвращает список товаров в формате JSON.
     - POST /items – добавляет новый товар и возвращает информацию о нем.
   - Для хранения товаров используйте список внутри приложения.

### Задание 3: Работа с JSON

3. Напишите программу на Python, которая:
   - Загружает данные из JSON файла (data.json),
    содержащего информацию о книгах (название, автор, год издания).
   - Выводит на экран имена всех авторов, где год издания больше заданного значения,
    которое вводит пользователь.

### Задание 4: Обработка JSON в Python

4. Напишите программу, которая:
   - Создает словарь с информацией о пользователе (имя, возраст, email).
   - Сериализует его в JSON формат и сохраняет в файл (user.json).
   - Затем задействуйте десериализацию для загрузки данных обратно из файла и выводите их на экран.

### Задание 5: Обработка ошибок при запросах

5. Напишите программу, которая:
   - Отправляет GET-запрос к некорректному URL и обрабатывает исключение.
   - Выводит на экран сообщение об ошибке, если запрос не удался.
